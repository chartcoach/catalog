## Use category-specific visual metaphors <!-- role: advice -->

Use iconic symbols when the symbol must suggest a specific kind of uncertainty, not just an ordered amount. For example, use a target-like metaphor for spatial accuracy, an hourglass-like metaphor for temporal precision, or a stoplight-like metaphor for attribute trustworthiness when readers need the symbol itself to cue the uncertainty category.

## Why iconic metaphors help category recognition <!-- role: reason -->

Iconic symbols can prompt a direct association between the mark and the uncertainty concept. That makes the symbol set feel more logical for category-specific uncertainty, although readers usually need more time to interpret it.

**Mechanism:** A fitted metaphor helps readers match the sign to the intended uncertainty type instead of relying only on abstract ordered variation.

**Evidence:** Across the category-specific uncertainty series, iconic symbol sets received slightly higher intuitiveness ratings overall than abstract symbol sets, while also taking longer to judge. The paper concludes that iconic symbols can be more intuitive when the reader understands both the uncertainty aspect and the metaphor on which the symbol is based [@maceachrenVisualSemioticsUncertainty2012].

**Notes:** The intuitiveness advantage was not consistent in every individual uncertainty series.

## Use when the uncertainty type itself must be recognized <!-- role: context -->

- **User Goal:** Help readers recognize what kind of uncertainty is being shown.
- **Task:** Identify or interpret a specific uncertainty category from the symbol set.
- **Data:** Uncertainty categories differ qualitatively, such as accuracy, precision, or trustworthiness.
- **Chart Setting:** Discrete point-symbol displays where the symbol can carry a metaphor.
- **Audience:** Readers who understand the uncertainty concept and can interpret the chosen metaphor.
- **Success Criterion:** The symbol-category match feels logical to readers.

## Do not use when speeded visual aggregation is the main task <!-- role: exceptions -->

**Break it when:** Readers must quickly compare aggregate uncertainty across many symbols in one display. **Why:** Abstract symbols supported faster grouped judgments.

## Know the tradeoff of iconic symbols <!-- role: costs -->

**Sacrifice:** You give up some speed.
**Risk:** A weak or unfamiliar metaphor can add confusion instead of clarity.
**Mitigation:** Test whether the specific pictorial metaphor is judged as logical before deploying it widely.

## Avoid unsupported or opaque metaphors <!-- role: mistakes -->

**Mistake:** Using an iconic symbol whose metaphor is not clearly tied to the uncertainty type. **Why it fails:** Iconic symbols only help when readers understand both the uncertainty aspect and the metaphor.

## Check whether the metaphor is actually working <!-- role: check -->

**Failure Sign:** Readers hesitate or disagree about the meaning of the iconic symbol set.
**Quick Check:** Show the iconic set alongside the named uncertainty category and compare its logic rating against an abstract alternative.
**Stronger Test:** Keep only iconic sets that score higher on intuitiveness without unacceptable delay for the intended task.

## Fix a weak iconic uncertainty symbol set <!-- role: fix -->

- Replace the current picture with a more recognizable metaphor for the same uncertainty category.
- Compare the revised iconic set against an abstract set before committing to it.
- If the metaphor remains weak or slow for the task, switch back to an abstract symbol set.
